# kchos0.dev

A minimalist personal portfolio and writing space built with Astro 5 and TinaCMS.

## Features

- Astro 5: High-performance static site generation.
- TinaCMS: Local-first content management.
- Minimalist Design: Typography-driven UI with dark mode support.
- Content Collections: Type-safe management for writing and projects.
- Modular Architecture: Organized components and utilities.

## Tech Stack

- Framework: Astro
- CMS: TinaCMS
- Styling: Vanilla CSS

## Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kchos0/kchos0-dot-dev.git
   cd kchos0-dot-dev
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server with the integrated TinaCMS admin:

```bash
npm run dev
```

- Site URL: `http://localhost:4321`
- CMS Admin: `http://localhost:4321/admin/index.html`

### Adding Content

Content can be managed through the TinaCMS interface or by adding Markdown files to `src/content/writing/`.
